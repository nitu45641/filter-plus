== Description ==

**[Filter Plus](https://woooplugin.com/filter-plus/)** is an **advanced WooCommerce filter plugin** that allows users to add multiple store filter like filter by category, product search, etc.

Ever wanted to provide better WooCommerce filtering on your WordPress eCommerce site? Now you can do that with **[Filter Plus](https://woooplugin.com/filter-plus/)** plugin. This will help users shorten the products quickly and order from the presized product list. Filter Plus will help you to improve the user experince, conversion rate while also helping the store owner reduce the bounce rate.

== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/filter-plus` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. All Settings will be found in Admin sidebar -> filter-plus
