How to Add Custom Post Type Filter

FilterPlus can filter any custom post type — portfolios, events, team members, and more. Follow the steps below using a Portfolio post type as a real example.

Step 1 — Register Your Custom Post Type

Add this code to your theme's functions.php or a custom plugin file. This registers a portfolio post type.

function register_portfolio_post_type() { register_post_type( 'portfolio', array( 'label' => 'Portfolio', 'public' => true, 'has_archive' => true, 'supports' => array( 'title', 'editor', 'thumbnail' ), 'taxonomies' => array( 'category', 'post_tag' ), ) ); } add_action( 'init', 'register_portfolio_post_type' );

After saving, the post type slug is portfolio — use this slug in FilterPlus.

Step 2 — Create a Filter Option

Go to FilterPlus → Filter Options

Click Add New to open the filter option form.

Set Filter Type to Custom Post Type

In the Filter Type dropdown, choose Custom Post Type. Enter portfolio as the post type slug. Enable the filters you need (Category, Tag, Author, etc.) and click Save.

Step 3 — Add the Filter to a Page

Shortcode

Paste this shortcode into any page, post, or text widget. Replace portfolio with your post type slug.

[wp_filter_plus filter_type="custom" custom_post="portfolio" template="1" no_of_items="9"]

Optional parameters:

show_categories="yes" show_tags="yes" author="yes" no_of_items="9" template="1"

Gutenberg Block Editor

1. Open the page in the block editor.

2. Click + and search for WordPress Content Filter.

3. Add the block. In the right-side panel, set Filter Type to Custom Post Type and enter portfolio as the slug.

Page Builder — Elementor

1. Edit the page with Elementor.

2. In the widget panel, search for WordPress Content Filter (under the FilterPlus section).

3. Drag it onto the page. In the Content tab, set Filter Type → Custom Post Type and enter portfolio.

Page Builder — Bricks Builder

1. Edit the page in Bricks.

2. In the elements panel, open the filter plus category and add Wordpress Filter.

3. In the element controls, set Filter Type → Custom Post Type and enter portfolio as the slug.
